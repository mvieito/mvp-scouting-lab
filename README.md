# MVP Scouting Lab ⚽
Plataforma de Inteligencia de Datos aplicada al Scouting de Fútbol.

## Características
- Ranking dinámico con ponderación por posición.
- Comparador avanzado de jugadores (Radar Charts).
- Generador de Once Ideal táctico.
- Exportación de informes en CSV.

## Instalación
`pip install -r requirements.txt`
